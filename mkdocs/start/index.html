入门

概述

MkDocs是python的第三方库，适用于文档和静态博客网站。文档源文件以Markdown编写，编写完毕可以快速生成静态网页并部署到github。

如果有条件可以用自己的域名而后用cdn加速。

这是项目完成后的样子，先一睹为快吧

下面正文开始...

安装MkDocs

查看python版本，没安装的同学需要先安装python3

$ python3 --version  
Python 3.8.4

安装MkDocs 

pip install mkdocs

检查mkdocs是否安装成功

$ mkdocs --version
mkdocs, version 1.1.2

创建Mkdocs

$ mkdocs new mkdocs-markdown
$ cd mkdocs-markdown

可以看到新创建的markdown目录

\----mkdocs-markdown\
    |----mkdocs.yml
    |----docs\
    |    |----index.md

有一个名为mkdocs.yml的配置文件，以及一个名为docs的文件夹，它将包含你的文档源文件。现在docs文件夹只包含一个名为index.md的文档页面。

MkDocs附带一个内置的开发服务器，可以让你在处理文档时预览文档。 确保与mkdocs.yml配置文件位于同一目录中，然后通过运行mkdocs serve命令启动服务器：

$ mkdocs serve
INFO    -  Building documentation... 
INFO    -  Cleaning site directory 
INFO    -  Documentation built in 0.05 seconds 
[I 201011 22:56:13 server:335] Serving on http://127.0.0.1:8000
INFO    -  Serving on http://127.0.0.1:8000
[I 201011 22:56:13 handlers:62] Start watching changes
INFO    -  Start watching changes
[I 201011 22:56:13 handlers:64] Start detecting changes
INFO    -  Start detecting changes

在浏览器中打开http://127.0.0.1:8000/，你将看到显示的默认主页：

更改logo

现在尝试编辑配置文件：mkdocs.yml。 将site_name设置更改为“Mkdocs教程”并保存文件。

site_name: My Mkdocs

此时logo的名字已改成My Mkdocs

更改Favicon图标

在docs文件下创建img文件夹，放入要更换的favicon.ico, 目录结构为：

----mkdocs-markdown\
    |----mkdocs.yml
    |----docs\
    |    |----img\
    |    |    |----favicon.ico
    |    |----index.md

可以看到favicon.ico已经更改成功

添加页面

  • 在docs目录下创建about1.md和about1.md两个文件
  • 修改mkdocs.yml

由于我们的文档站点将包含一些导航，你可以编辑配置文件，并通过添加nav设置来管理每个页面导航的顺序、标题和嵌套情况：

site_name: My Mkdocs
nav:
    - Home: index.md
    - About:
        - About1: about1.md
        - About2: about2.md

更改样式

在docs文件夹下创建extra.css文件 目录结构为：

----mkdocs-markdown\
    |----mkdocs.yml
    |----docs\
    |    |----extra.css
    |    |----img\
    |    |    |----favicon.ico
    |    |----index.md
    |    |----about1.md
    |    |----about2.md

找到导航选中后的类覆盖为:

.navbar-dark .navbar-nav .active>.nav-link,
.navbar-dark .navbar-nav .nav-link.active{
    color: #02294E;
}

在mkdocs.yml添加extra_css: [extra.css]

site_name: My Mkdocs
nav:
    - Home: index.md
    - About:
        - About1: about1.md
        - About2: about2.md
extra_css: [extra.css]

此时选中导航后字体颜色变成了黑色

部署

安装github

mkdocs需要部署到github，首先安装

$ pip install git

检查git是否安装成功

$ git --version
git version 2.28.0
  • 新建项目
  • 选择join for free
  • 填写相应的信息
  • 在刚创建的项目新建一个文件夹
  • Repository name这一栏填写xx.github.io,其中xx是你的项目名称

克隆项目到本地

创建成功后克隆项目到本地git clone xx，其中xx是你的项目地址

$ git clone https://github.com/MkdocsDemo/mkdocsdemo.github.io.git

打包mkdocs

$ cd mkdocs-markdown
$ mkdocs build
INFO    -  Cleaning site directory 
INFO    -  Building documentation to directory: /mkdocs-markdown/site 
INFO    -  Documentation built in 0.06 seconds

复制site下的所有文件到前面在git克隆的项目

把文件添加到仓库

$ cd mkdocsdemo.github.io
$ git add --all
